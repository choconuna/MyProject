## 개요
* 프로젝트 이름: 멍노트(Mung Note)
* 프로젝트 제작 기간: 2023.03-2023.05
* 개발 인원: 이소진(1인 개인 프로젝트)

## 개발환경
* 개발 언어: Kotlin
* DB: Firebase
* IDE: Android Studio
* API, 라이브러리: RESTful API, Retrofit, OpenWeatherMap API, NaverMap API
